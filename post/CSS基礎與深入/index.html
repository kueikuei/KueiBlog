CSS元素位置-(相對/絕對)定位

分為四種定位模式:

  1. static - 預設定位位置，以左上角原點為起始位置，是最固定無法更動的。
  2. relative - 相對預設排列位置，相較於static，是可以變移動的。
  3. absolute - 最自由最容易移動，可以輕易指定到任何位置。
  4. fixed - 相較於absolute，不會隨著畫面滾輪移動而消失，常用於導覽列(Navbor)。

CSS內間隔/外間隔(padding/margin)

CSS 排版有一個很重要的觀念：Box Model 。它描述了元素之間的彼鄰關係，同時也左右了我們是否能夠成功透過CSS，完成整個頁面的呈現。
