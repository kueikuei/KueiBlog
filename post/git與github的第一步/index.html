前言

經常在寫程式時遇到需要恢復原本程式狀態軟體發⾏，需要分成維護版(released)跟開發版(development)，那何謂版本控制呢？

版本控制

  1. 可以隨時復原修改，回到之前的版本
  2. 多⼈協作時，不會把別⼈的東⻄蓋掉
  3. 保留修改歷史記錄，以供查詢
  4. 軟體發⾏時，可以⽅便管理不同版本

Git與Github

What is git?
一個開放源碼的版本管理系統(VCS)
發明⼈ Linus Torvalds (Linux creator)
證明了厲害的工程師同時也是軟體發明者
git無論在local或remote都能進行版本控制

What is github?
一個基於 git 開發出來的服務
讓你能透過 git 在遠端備份程式碼
一般的使用免費但必須公開程式碼
廣受大量開源專案的喜愛
是程式設計師的寶庫 (可以在裡面查詢各式各樣的程式與實作的程式碼)
