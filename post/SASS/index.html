CSS與SASS比較

css

sass

  • 加入變數功能(key:value形式)
  • 當要大量改變某變數屬性時相當方便
  • 可以用&表示元素：看起來更有層次感，更能了解是什麼元素包在什麼裡頭
  • minin：CSS的function，下一節會特別說明

看起來是否更直觀簡單呢？

mixin

可否聽說css有function呢？直呼其名為mixin

mixin的作用在於讓我們將常用的CSS指令是先定義好，當程式project寫大時，只要改一個屬性，就能調整所有相關CSS。

mixin也經常結合sass變數，將常用的元件模組化，例如尺寸、文字。

完整範例請見Mixin

SASS系統化CSS Style

在寫一品牌網站時會先定義後該有的佈局、字體顏色等等，而透過SASS的變數、mixin、css排列能夠更加系統化地去完成網頁，例如改變一個變數顏色、一個變數size，整份網頁對應到的變數都能有所改變，而當網頁系統相當龐大時此方法就會相當便利。

而透過此範例也能更加瞭解CSS在佈局定位時的技巧。
