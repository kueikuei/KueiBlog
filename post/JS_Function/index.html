Scope Chain

用var宣告的變數，作用範圍在當時的環境(函數內);沒有用var宣告則是全域物件(window)的屬性，也就是全域範圍。
scope chain特性，JS在查找變數時，會循著範圍練一層層往外找，函數內找不到則往函數外找
內部函式可以存取外部函式變數，外部則無法存取內部。

###Self-Invoking Function

先解讀裡面的匿名函式後，後面再加個()去執行它。

Closure

Execution Context和其外面的變數關閉（Close）在一起，而這樣的現象就稱為closure
能將外部函式的變數關到內部函式，用途是延續外部變數的生命
閉包所關起來的是變數的reference，而不是變數的value
