環境安裝

  • brew update # 更新 brew
  • brew install mongodb # 安裝 MongoDB
  • brew upgrade mongodb # 更新 MongoDB
  • mongod 或加sudo mongod # 啟動 Local端MongoDB Server
  • mongo # 連線到 localhost:27017 ，不需帳密。
  • 27017 預設 port

  • 預設目錄在:/usr/local/Cellar/mongodb/3.4.0/data/db

  • mkdir -p /data/db # 建立資料庫儲存的目錄。

  • chown id -u /data/db # 授權當前使用者目錄權限。
  • id -u # 列出當前使用者 uid
  • mongod –dbpath /data/db # 設定資料庫資料目錄。
  • Robomongo:快速檢視資料庫中資料

基本指令(Mongo Shell)

終端機輸入mongo啟動Mongo Shell
資料庫中的CRUD

  • db 當前使用的資料庫。
  • show dbs 資料庫列表。
  • show collections 資料表清單。
  • use [dbname] 切換使用目標資料庫。
  • db.getSlibingDB(‘dbname’) 取得資料庫參考物件。
  • db.dropDatabase() # 移除資料庫。
  • db.addUser( { user: ““, pwd: ““, roles: [] } ) # 加入使用者 # db.addUser is not a function
  • db.auth(‘id’, ‘pwd’) # 登入認證身份。
  • printjson(obj) 將物件以 JSON 格式輸出。
  • db.createCollection(name, {capped: , autoIndexId: , size: , max } ) # 建立集合，主要的功能是用來開固定式集合，固定式集合不能刪除 Row，但可以移除整個資料表，一旦 capped 設為 true 就要指定 size，但如果要限制則必須在指定 max 。一旦超出 max 舊的資料就會被覆寫。
  • db.collection.drop() # 移除資料表。
  • db.collection.ensureIndex({name: [1|-1]}) # 建立索引
  • db.collection.getIndexes() # 取得索引
  • db.collection.dropIndex({name: 1}) # 移除索引
  • db.testData.find() # 檢視資料
