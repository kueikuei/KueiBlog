========================

原生物件介紹

string

與array的差別在於string用了method後不會改變原始變數

array

能動態新增元素

date

取得電腦日期和時間

math

不需要 new 運算子自己就會 建立，故可以直接使用屬性和方法

error

Error 物件會儲存執行時發生錯誤的資 訊，發生錯誤時會自動建立

regexp

使用正規表示法的物件，用以比對其它字串是法符合範本字串
