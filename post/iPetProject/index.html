主程式碼有index.html、main.js、gulpfile.bebel.js三個。

  • index.html

將main.js撈取到的資料整理好後最後放進<div class="col-md-12" id="data-list">中

  • main.js
    可說是整個project的核心檔案，可以將其拆成三個部分去探討-變數定義、資料撈取、塞入前端。

變數定義

首先將open data的json網址篩入變數REQUEST_URL，並且製造兩個容器，分別是data與str，data是之後存放資料的容器、而str是前端的容器。

資料撈取

這邊利用fetch()進行資料的撈取，一但得到資料則promise回傳物件，透過then接續做事，要注意的是回傳的資料結果此時為response參數，並將其轉成json格式，再進一步return給後續的then func，最後篩入容器data中並呼叫dataList()做接下來的動作。

塞入前端

得到的資料結構此時是這樣的：
data = [{},{},{}]
故這邊透過迭代方法map()將資料一筆筆抽出來放進參數key，此時就可以呼叫各筆資料裡頭的屬性，然而有個問題在於要如何將這邊變數塞進前端？別擔心，ES6提供了簡單的方法${var}，只要以這種形式就能將變數塞入，有寫過jade的讀者應該不陌生，最後透過選擇棄選到index.html的data-list並將資料塞入便完成。

心血來潮，來提一下bootstrap吧!!!要知道，好的商品也需要好的包裝，bootstrap就是在做這件事情。

這邊的col-sm-6、col-md-4其實都是在做RWD(Responsive web design)，讓各種尺寸的介面都能得到最好的視覺體驗，而thumbnail-image是照片模式的其中一種呈現方式，其他還有圓形.img-circle、橢圓邊角.img-rounded等等，透過bootstrap可以輕鬆呈現出這些好看的結果。

  • gulpfile.bebel.js
    Gulp簡單來說，就是個自動化任務的工具。比如說開發網頁的時候，想要在儲存檔案時自動重整網頁，那就設定好，讓Gulp幫你執行；想要把css、js檔都minify，寫好讓Gulp幫你執行；想要……，只要找到適當的模組，Gulp都可以幫你執行([程設雜筆] 設定ReactJS + Gulp)。
