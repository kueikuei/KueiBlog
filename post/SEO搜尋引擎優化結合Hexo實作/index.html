SEO為何

SEO(Search Engine Optimization)，中文翻譯為搜尋引擎優化，搜尋引擎最佳化等等。

今天費盡千辛萬苦，將網頁寫好並且放到雲端上，假若不能被搜尋到，一切都是枉然。

沒錯，這正是SEO重要性所在。而SEO運用一系列的方法，讓搜尋引擎看懂你的網站內容，然後使你的網站排名出現在自然搜尋結果的前面，進而取得高流量。而SEO的最終目標是讓網站排在第一頁，排名越前面越好(SEO是甚麼－搜尋引擎優化概要)。

相關技巧有許多，例如：有沒有善用HTML5重要Tag、RWD有無、是否加入廣告等等，有興趣的讀者可以再詳閱SEO是甚麼－搜尋引擎優化概要。

而本文主要是期望能透過有名的Hexo Blog讓Google Search Engine能夠更容易發現我們的Blog。

那麼……開始囉~~

Google Webmaster Central 結合 Hexo

要了解，每一個Search Engine都有自己的搜尋法演算法與特性，今天要盡快讓一個搜尋引擎搜尋到自己的文章，最快的方式是主動出擊提交自己的站點。

  • Google



  • 百度
  • Bing

而當有大量文章時，一篇篇提交就顯得很麻煩。

透過Google搜索引擎提交Sitemap

步驟如下：

  1. 進入頁面並將自己的Blog首頁匯入
  2. 進入到網頁驗證處
  3. 跟著網頁提示步驟進行
  4. 跟著步驟1.下載HTML Verification
  5. 將剛下載的ver code插入hexo的_config.yml
  6. 重新hexo g後產生google-site-verification.xml file
  7. 將整個Blog project重新推到雲端上
  8. 透過步驟3.進行驗證
  9. 驗證成功後依序步驟4.將整個驗證完成
  10. 為部落格產生sitemap
  11. 將sitemap:
    path: sitemap.xml插入_config.yml
  12. 重新hexo g後產生sitemap.xml file
  13. 將整個Blog project重新推到雲端上
  14. 進入Sitemap提交站提交sitemap.xml後便完成
  15. 等待google驗證
